# secure-static-website-aws
Secure static website hosted on AWS S3 + CloudFront with OAC and WAF

##  🚀 Architecture
- Amazon S3(Private Bucket)
- Cloudfront(CDN)
- Origin Access Control(OAC)
- AWS WAF(Security)

##  🔐 Features
- HTTPS Enabled
- No public S3 access

- Security Headers
- DDoS protection via WAF

- AWS Cloudfront Setup
- Secure S3 access using OAC
- WAF Configuration
