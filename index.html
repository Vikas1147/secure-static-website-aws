About This Project

This project demonstrates a secure, scalable static web architecture built on AWS using modern best practices. The goal was to design a high-performance and fault-tolerant system while ensuring strong security.

🚀 Architecture Overview

  • Amazon S3 (Private Bucket) for storing static website files
  • CloudFront CDN for global content delivery and caching
  • Origin Access Control (OAC) to restrict direct S3 access
  • AWS WAF to protect against web attacks (SQL injection, XSS, bots)

🔐 Security Features

  • Blocked all public access to S3 bucket
  • Access allowed only through CloudFront using OAC
  • HTTPS enforced using CloudFront
  • Security headers applied for browser-level protection

⚡ Performance Optimization

  • Global CDN caching using CloudFront
  • Reduced latency and faster load times
  • Optimized static content delivery

🧠 Key Learnings

  • Difference between public and private S3 hosting
  • Using OAC for secure origin access
  • How CDN caching works in CloudFront
  • Implementing AWS WAF for real-world security
